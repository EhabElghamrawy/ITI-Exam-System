ITI Examination Management System

Project Overview
This is a comprehensive SQL Server database designed to manage the examination process within an educational institute. It covers everything from student enrollment to automated exam generation and grading.

Key Features
- Relational Design:Organized into Tracks, Intakes, Courses, and Instructors.
- Automated Exams:Uses Stored Procedures to generate randomized exams for students.
- Result Tracking:Tables to store student answers and calculate final grades automatically.
- Data Integrity:Solid relational structure with primary and foreign key constraints.

Getting Started
1. Open SQL Server Management Studio (SSMS).
2. Right-click on **Databases** -> **Restore Database**.
3. Select the ITI.bak file provided in this repository.
4. Execute the Final Project Query.sql to explore the schema and procedures.
